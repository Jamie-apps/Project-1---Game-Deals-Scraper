<h1>Project 1: Game Deal Scraper</h1>

<p>A lightweight and efficient web application that automatically scans multiple vendors to find the best deals on video games, helping gamers save time and money.</p>

<p><strong>Tools & Technologies:</strong></p>

<ul>
<li>Python – Core programming language</li>
<li>HTML - Front-end language</li>
<li>BeautifulSoup – HTML parsing and data extraction</li>
<li>Flask – Web framework for serving results</li>
<li>Cheapshark API - game deal source </li>
<li>RAWG API- Game genre searcher </li>
</ul>

<p><strong>Features:</strong></p>

<ul>
  <li>- Search games by title</li>
  <li>- Filter by price, genre and store</li>
  <li>- Sort by discount, price or alphabetically</li>
  <li>- "Deal of the day" ranking system</li>
  <li>- CSV export of filtered reports</li>
  <li>- Pagination system</li>
  <li>- Animated expandable deal cards</li>
  <li>- Retro arcade UI</li>
</ul>
